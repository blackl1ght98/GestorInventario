                    /*Swith de control de flujo vs swith de expresion con ejemplos:
                    Switch de control de flujo:
                    int day = 3;
                        string dayName;
                        switch (day)
                        {
                            case 1:
                                dayName = "Monday";
                                break;
                            case 2:
                                dayName = "Tuesday";
                                break;
                            case 3:
                                dayName = "Wednesday";
                                break;
                            default:
                                dayName = "Unknown";
                                break;
                        }
                    Un switch de expresión es una característica introducida en C# 8.0 que permite evaluar una expresión y devolver un valor basado en 
                    varios patrones. A diferencia del switch de control de flujo, que es una declaración, el switch de expresión es una expresión en 
                    sí misma, lo que significa que produce un valor y puede usarse en asignaciones y otros contextos donde se espera un valor.
                    
                    int day = 3;

                    string dayName = day switch
                    {
                        1 => "Monday",
                        2 => "Tuesday",
                        3 => "Wednesday",
                        _ => "Unknown"
                    };
                    Diferencias Clave

                    Declaración vs. Expresión:
                    El switch de control de flujo es una declaración que no devuelve un valor y se utiliza principalmente para ejecutar bloques 
                    de código en función de una condición. El switch de expresión es una expresión que evalúa y devuelve un valor basado en patrones.

                    Sintaxis:
                    El switch de control de flujo utiliza case y break para definir bloques de código.
                    El switch de expresión utiliza => para asociar valores con resultados y no requiere break.

                    Uso:
                    El switch de control de flujo se usa en situaciones donde se necesita ejecutar diferentes bloques de código.
                    El switch de expresión se usa cuando se necesita evaluar una condición y obtener un valor basado en esa evaluación de manera concisa.
                    --------------------------
                    Esto para el swith de expresion--> En base a patrones que le pasamos devuelve un resultado u otro le pasamos como valor 3 y dentro de 
                    los patrones disponibles devuelve lo que vale 3 que es wednesday.
                    Esto es para el switch de control de flujo--> similar al anterior pero es como algo asi case 1 es igual a 3 no pues pasa a la siguiente
                    case 2 es igual a 3 no pues pasa a la siguiente, case 3 es igual a 3 si pues devuelve el valor que es Wednesday.

                    */
